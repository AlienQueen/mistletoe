Mistletoe Project

Mistletoe is a junit extension intended for integration testing. In technical terms, mistletoe is a junit test suite runner presenting its test results on a web-page.

Application which run perfectly in the developer's environment may not run properly in the production environment. Such problems are often due to configuration errors and connectivity issues. Intead of perusing log files, or looking at stack traces, mistletoe offers a convenient method for diagnosing integration problems. Given a test suite checking for integration issues such as configuration errors, connectivity to your database or to neighboring applications, you can home in on integration problems quickly and conveniently.

See the user manual for details.
