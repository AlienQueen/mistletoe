Dependencies per module

Each mistletoe module has a different set of dependencies. These are listed below in a separate table per module.

mistletoe-core

  • junit, tested with version ${junit.version}

mistletoe-servlet

  • mistletoe-core and its dependencies.
  • javax.servlet.servlet-api version ${servlet-api.version}

mistletoe-wicket

  • mistletoe-core and its dependencies
  • wicket version ${wicket.version}
