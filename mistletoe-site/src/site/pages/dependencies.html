Dependencies per module

Each mistletoe module has a different set of dependencies. These are listed below in a separate table per module.

mistletoe-core

  • junit , tested with version 4.7

mistletoe-servlet

  • mistletoe-core and its dependencies.7
  • javax.servlet.servlet-api version 2.5

mistletoe-wicket

  • mistletoe-core and its dependencies.7
  • wicket version 1.4.2
  • slf4j-api version 1.5.8
